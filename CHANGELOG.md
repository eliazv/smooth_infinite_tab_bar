- Added animated selection indicator (scale + fade).
- Added optional sublabel and highlight dot per item.
- Added overflow arrow buttons with auto-show/hide animation.
- Added `MultiDirectionalInfiniteScroll` as a standalone exported widget.
- Added example app with month, week and finite category selectors.
